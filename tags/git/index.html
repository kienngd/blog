This is some git commands that I usually use. Clone a repository git clone <URL> [<DEST Directory>] Clone from specific branch git clone --branch <BRANCH NAME> <URL> [<DEST Directory>] Pull code from remote master branch and merge to current banch git pull origin master Push code from local DEV branch to remote MASTER branch git push origin dev:master Create local branch git checkout -b <BRANCH NAME> Switch local branch git checkout <BRANCH NAME> List all branch
