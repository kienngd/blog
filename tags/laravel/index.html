Home page: http://docs.apiato.io/ Create project using composer from latest stable composer create-project apiato/apiato apiato9 Then edit the .env file. Migrate database php artisan migrate Create admin user, some permissions, roles php artisan db:seed Default account is: admin@admin.com / admin. Add all current permissions to admin role. php artisan apiato:permissions:toRole admin Install OAuth 2.0 php artisan passport:install Then edit file .env update CLIENT_WEB_ADMIN_ID, CLIENT_WEB_ADMIN_SECRET with the new created value.
