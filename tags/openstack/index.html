Working with openstack api using curl

Kien Nguyen - Oct 29, 2020
Categories:  notes
Tags:  PHP     OpenStack     API    
Get token curl -s -i -H "Content-Type: application/json" -d ' { "auth": { "identity": { "methods": [ "password" ], "password": { "user": { "name": "{USERNAME}", "domain": { "id": "{DOMAIN_ID}" }, "password": "{PASSWORD}" } } }, "scope": { "project": { "id": "{PROJECT_ID}" } } } }' "https://{BASE URL}:5000/v3/auth/tokens" | grep "X-Subject-Token:" Default keystone port is 5000 By default, this token will be expired after 6 hours. Export token API_TOKEN="{TOKEN}" Get domains curl -s -H "X-Auth-Token: $API_TOKEN" "https://{BASE URL}:5000/v3/domains" | jq '.
