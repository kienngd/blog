Use UUID as primary key of Laravel Eloquent ORM

     Kien Nguyen -    2020-11-02
    howto
    Laravel     UUID    

Install package using composer:

composer require goldspecdigital/laravel-eloquent-uuid:^7.0

Inside migration file.

// Replace this
$table->increments('id');
// By this
$table->uuid('id')->primary();
